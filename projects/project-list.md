## LRU Rules
1. New projects start at position #1
2. Max 10 active per type — position #11 auto-archives
3. Loading a project moves it back to #1
4. Archived projects can be reloaded anytime

## Commands
- `new [type] project [name]` — Create new project
- `load project [name]` — Resume existing project
- `save project` — Save current project progress
- `list projects` — Show all projects
- `archive project [name]` — Manually archive

*Types: coding, writing, research, business*
